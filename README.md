# MERN Portfolio Project

A full-stack portfolio website built using React and Node.js.

## 🚀 Features

* Personal portfolio UI
* Projects showcase
* Contact form with email functionality
* Responsive design

## 🛠 Tech Stack

* Frontend: React, CSS
* Backend: Node.js, Express
* Email: Nodemailer / EmailJS

## 📦 Installation

```bash
git clone https://github.com/your-username/portfolio-mern-project.git
cd Mern-Portfolio-Project
npm install
cd client
npm install
npm start
```
